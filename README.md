# Products CRUD API

## Getting Started

### 1. Clone repo

```
git clone https://github.com/pratikmane1299/PRODUCTS-CRUD-API.git
```

### 2. Install Dependencies

```
npm install
```

### 3. Start the server

```
npm run dev
```

## Other Commands

### Lint

```
npm run lint
```

### Test

```
npm run test
```
